# Python Basic Programs

This repository contains 10 beginner-friendly Python programs that demonstrate fundamental programming concepts such as input/output, conditions, loops, strings, lists, and basic mathematics.

## Programs Included

### 1. Hello World
Prints the message:
```
Hello, World!
```

### 2. Add Two Numbers
- Takes two integer inputs from the user.
- Displays their sum.

### 3. Even or Odd
- Accepts an integer.
- Checks whether the number is even or odd.

### 4. Largest of Two Numbers
- Takes two numbers as input.
- Displays the larger number.

### 5. Factorial
- Accepts a positive integer.
- Calculates and displays its factorial using a loop.

### 6. Multiplication Table
- Takes a number as input.
- Prints its multiplication table from 1 to 10.

### 7. Prime Number
- Checks whether the entered number is prime or not.

### 8. Reverse a String
- Accepts a string from the user.
- Prints the string in reverse order.

### 9. Maximum in a List
- Uses a predefined list.
- Finds and displays the largest element using `max()`.

### 10. Fibonacci Series
- Accepts the number of terms.
- Prints the Fibonacci sequence up to the specified number of terms.
