# Multi-Utility Toolkit

A Python-based command-line application that provides a collection of useful utilities, including date and time operations, mathematical calculations, random data generation, UUID creation, file handling, and Python module exploration.

## Features

### 1. Datetime and Time Operations
- Display current date and time
- Calculate the difference between two dates
- Format dates into multiple formats
- Stopwatch
- Countdown timer
### 2. Mathematical Operations
- Calculate factorial
- Area calculation for:
  - Circle
  - Rectangle
  - Triangle
  - Square
- Compound Interest Calculator *(requires implementation)*
- Trigonometric Calculations *(requires implementation)*
### 3. Random Data Generation
- Generate random numbers
- Generate random lists
- Create secure random passwords
- Generate OTPs
- Random sampling from a population
### 4. UUID Generator
- Generate UUID Version 1
- Generate UUID Version 4

### 5. File Operations
- Create a file
- Write to a file
- Read from a file
- Append data to a file

### 6. Module Explorer
- Explore any installed Python module using `dir()`
## Requirement
- Python 3.8 or higher
No external libraries are required. The project only uses Python standard library modules:
- math
- random
- string
- time
- uuid
- datetime
- importlib
